Projectile Physics Simulation (CLI)

A lightweight, high-performance Python engine developed to simulate 2D kinematics specifically for mobile-constrained environments (Termux/Linux).

🚀 Features
High Precision: Implements standard kinematic equations for 
time-of-flight,
peak height, and horizontal range.

Mobile Optimized: Designed as a headless CLI tool for fast execution on Android/Termux setups.

A/L Physical Science Focus: Built to verify manual mechanics calculations for the GCE Advanced Level curriculum.

Visual Feedback: Includes a text-based trajectory renderer for terminal environments.

🛠 Usage
Ensure you have Python installed, then run: python projectile.py
